# PHP Native - Interpersonal CRUD

Aplikasi CRUD sederhana menggunakan **PHP Native** sebagai latihan dasar pengelolaan data tanpa framework.  
Proyek ini berisi fitur dasar untuk menampilkan, menambah, mengedit, dan menghapus data dari database MySQL.
Didalam folder css isinya sama, perbedaannya hanya menggunakan Boostrap
